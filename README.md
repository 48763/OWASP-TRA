### A1 Injection

注入（Injection）漏洞，像 SQL、OS、XXE 和 LDAP 注入，發生在不可信任的數據（資料）傳送至直譯器，並作為指令或查詢的一部分。
攻擊者的惡意資料（數據）可能會導致直譯器（interpreter）直接執行未授權的指令或存取資料。
